Tentang Array

Array adalah struktur data linier yang menyimpan elemen dalam lokasi memori yang berurutan. Setiap elemen dapat diakses menggunakan indeks.

Karakteristik Array

  • Elemen disimpan dalam memori yang berurutan
  • Akses acak dengan kompleksitas O(1)
  • Ukuran tetap (array statis)
  • Tipe data homogen

Operasi Dasar

  • Insert: Menambahkan elemen ke dalam array
  • Delete: Menghapus elemen dari array
  • Search: Mencari elemen di dalam array
  • Access: Mengakses elemen berdasarkan indeksnya
