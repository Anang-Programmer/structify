Tentang Linked List

Linked List adalah struktur data linier di mana elemen disimpan dalam node. Setiap node berisi data dan pointer yang menunjuk ke node berikutnya.

Karakteristik Linked List:

  • Ukuran dinamis - ukurannya dapat berubah saat program berjalan
  • Akses sekuensial - diakses secara berurutan dari head
  • Alokasi memori tidak berurutan
  • Penyisipan dan penghapusan yang efisien

Operasi Dasar:

  • Insert: Menambahkan node baru (awal, akhir, posisi tertentu)
  • Delete: Menghapus node (awal, akhir, berdasarkan nilai)
  • Search: Mencari nilai dalam daftar
  • Display: Menampilkan semua elemen
